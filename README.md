# ⚖️ CORE Engine: A Financial Stability Web App

## Project Overview
🔹 This dashboard is a financial tool designed for Snohomish County residents to visualize the "Stability Gap." <br>
🔹 It utilizes the **CORE (Creating Open Roads to Equity)** framework <br>
🔹 Helping families understand how one-time financial boons, like a VITA tax refund, bridge the gap toward self-sufficiency benchmark.

## Key Features
▫️ **Dynamic Stability Logic:** Calculates the monthly gap based on the University of Washington Self-Sufficiency Standard.<br>
▫️ **Visual Analytics:** A color-coded progress bar (Red/Yellow/Green) provides instant feedback on financial sufficiency.<br>
▫️ **Housing Security Metric:** Translates tax refunds into "months of rent" to provide a tangible security timeline.

## Data Methodology
The $10,000/month benchmark is sourced from the [UW Self-Sufficiency Standard](https://selfsufficiencystandard.org/washington/) <br>
Specifically for Snohomish County, WA.
